### 题目

[反转链表](https://leetcode-cn.com/problems/fan-zhuan-lian-biao-lcof/)

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    let pre = null;
    let cur = head;
    let next;
    while(cur) {
        next = cur.next;
        cur.next = pre;
        
        pre = cur;
        cur = next;
    }

    return pre
};
```

### 总结

算法：迭代

记录遍历结点 cur
记录上个结点 pre
记录下个结点 next

难点：指针的指向，容易混，指针操作顺序有要求。
