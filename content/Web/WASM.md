- WebAssembly
  - WebAssembly（缩写为 Wasm）是一种基于堆栈式虚拟机的二进制指令集。Wasm 被设计成为一种编程语言的可移植编译目标，并且可以通过将其部署在 Web 平台上，以便为客户端及服务端应用程序提供服务
    - 虚拟机-[计算模型分类](https://time.geekbang.org/column/article/283436)
      - **堆栈机**
      - 累加器机
      - 寄存器机
  - Wasm 核心标准（Core Interfaces）
  - API
    - MVP
    - WASI（POSIX syscall）
    - WCGI
