# 登录鉴权

- 登录鉴权
  - 会话机制
    - cookie(sessionID)-session
    - Session 的分布式问题
      - 独立的缓存服务器：共享 session
  - token 机制
    - token 使用方式
      - 存储
        - cookie
        - localstorege
      - 过期-刷新
      - 验证-防篡改：签名
    - [JWT](https://jwt.io/)
  - 单点登录：一次登录，全线通用
    - 核心：单独**认证服务**（通常被称为 SSO）
    - 顶级域相同：cookie 顶级域共享 + session
    - 顶级域不同


对 session 和 token 的对比本质上是「客户端存 cookie / 存别地儿」、「服务端存数据 / 不存数据」的对比。
