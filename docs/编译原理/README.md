文法是编译原理中对语言的写法的一种规定，一般来说，文法分成词法和语法两种。

1. 词法分析：状态机或者正则表达式
2. 语法分析
   1. LL LR
   2. 语法定义多数采用 BNF
      1. 语法定义的核心思想不会变，都是几种结构的组合产生一个新的结构，所以语法定义也叫语法产生式
