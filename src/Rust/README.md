- 基础
  - 程序结构
    - main
    - 语句以分号 ; 结尾
  - 变量
    - 声明
      - let
    - 不可变与可变
      - 变量绑定值后默认不可变
    - 变量隐藏  
      重复声明同一变量名，原变量不会再被访问到
  - 类型
    - 编译器通常可以根据绑定值推断变量的数据类型
    - 数字
      - 整数数字（默认 i32）
      - 浮点数
    - 布尔型
    - 字符 char
    - 集合
      - 字符串 &str
      - 元组
      - 结构体
        - 经典结构
        - 元组结构
        - 单元结构？
      - 枚举
      - 数组 `[T; size]`
      - 向量，向量的大小或长度可以随时增大或缩小
      - `vec![数组]`
  - 流程控制
    - 条件 if
  - 函数
    - 函数的定义位置无所谓
    - 返回值类型 `-> <type>`
    - 在 Rust 中，始终返回代码块 ({ ... }) 中最后一个表达式的值。 可以根据需要显式使用 return 关键字。
