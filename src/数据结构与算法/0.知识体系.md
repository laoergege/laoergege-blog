- 数据结构与算法
  - 复杂度
    - 时间复杂度
    - 空间复杂度
    
  - 数据结构（不同数据结构的差异在增删查效率上）
    - 储存结构
      
      底层结构决定上层操作
      
      - 顺序存储（数组）
      - 链式存储（链表）
      
    - 逻辑结构
    
      对数据操作进行逻辑限制
    
      - 线性表
      
          线性表就是数据排成像一条线一样的结构。每个线性表上的数据最多只有前和后两个方向
          - 数组
        - 链表
      
        - 非线性表
    
  - 算法
  
    - 查：对于大多数数据结构无非就是遍历 + 索引
      - 遍历
        - 线性就是 for/while 迭代为代表
        - 非线性就是递归为代表
